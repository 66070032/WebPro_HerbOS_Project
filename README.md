
# Herb OS

ทำการ Clone Project ไปยังเครื่องตัวเอง

```bash
  git clone https://github.com/66070032/WebPro_HerbOS_Project
```

แตกไฟล์ด้วยโปรแกรมแตกไฟล์ (WinRar / WinZip / 7Zip)

รันไฟล์ run.bat

เข้าใช้งานเว็บไซต์ด้วย URL: http://localhost:3000
